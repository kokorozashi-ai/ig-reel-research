# セットアップ手順書｜AI初心者でも45分で完了

このガイドは **初めて触る方でも順番にやれば必ず動く** ように作られています。 飛ばさず最初から順番に進めてください。

## 全体の流れ

```
✅ 用意するもの 4 つを揃える（45 分）
  1. Apify  → Instagram の情報を集めてくれるサービス（10 分）
  2. OpenAI → 文章を分析してくれる AI サービス（10 分）
  3. Google Cloud → スプレッドシート連携の仕組み（20 分）
  4. Google スプレッドシート → 結果を表示する場所（5 分）
✅ 取得した情報を Mac に登録する（10 分）
✅ 試しに動かしてみる（10 分）
```

> 🎓 **大事なポイント**：難しそうに見えても、 **指示された場所をクリックするだけ** で進めるように作っています。失敗しても何度でもやり直せます。

## Phase 0: 事前準備（5 分）

最初に Mac に必要なツールを 1 つだけ入れます。「Homebrew」というMac用のツール管理アプリです（既に入っていればスキップ）。

### Macの「ターミナル」アプリを開く

「ターミナル」とは、文字でMacに指示を出すアプリです。普段は使わない人も多いですが、**今回だけ使います**。

1. Mac の右上の **🔍（虫眼鏡）アイコン** をクリック
2. 「**ターミナル**」と入力 → Enter
3. 黒い画面が開く（これが「ターミナル」です）

### 必要なツールをインストール

ターミナルに以下を **コピペして Enter** してください（1 行ずつ）：

```bash
# 動画処理ツールを入れる（既に入っていれば「すでに最新です」と出ます）
brew install ffmpeg

# Python の関連ツールを入れる
pip3 install --break-system-packages openpyxl gspread google-auth requests openai openai-whisper "scenedetect[opencv]"

# 設定ファイル置き場を作る
mkdir -p ~/.config/gcloud

# 音声を文字に変換するAIモデルを事前ダウンロード（約1GB・数分かかります）
python3 -c "import whisper; whisper.load_model('medium')"
```

> 🛑 もし `brew not found` と出たら、Homebrew が入っていません。<https://brew.sh/index_ja> の手順で先に入れてください（5 分）。

## Phase 1: Apify トークンを取得（10 分・無料枠あり）

### Apify って何？

Instagram のリール動画情報を自動で集めてくれる **Web サービス** です。 月 5 ドル分（約 750 円）まで **無料で使える** ので、最初は無料枠内で十分です。

### 手順

1. 🌐 ブラウザで <https://apify.com/> を開く
2. 右上の「**Sign up**」（または「Get started」）をクリック
3. 「**Continue with Google**」を選んで Google アカウントでログイン（一番楽です）
4. 簡単な質問が出ますが、適当に答えるか「Skip」で飛ばしてOK
5. ログイン後、左メニュー一番下の **歯車アイコン → Settings**
   - もしくはこの URL に直接アクセス: <https://console.apify.com/settings/integrations>
6. 「**Personal API tokens**」のところで「**+ Create a new token**」をクリック
7. 入力画面：
   - **Description**：`ig-reel-research` と入力
   - **Set expiration date**：オフのまま
   - **Limit token permissions**：オフのまま
8. 「**Create**」をクリック
9. 表示された `apify_api_xxxxxxxxxxxxx...` という長い文字列を **必ずコピーしてメモ**

> 🛑 **大事**：このトークンはこの画面でしか表示されません。コピーし忘れたら、削除して作り直してください（やり直しOK）。

> 📸 [スクショ: `screenshots/1_apify_token.png`]

## Phase 2: OpenAI の鍵を取得（10 分・要クレカ）

### OpenAI って何？

ChatGPT を作っている会社の **AI サービス** です。集めたリール情報を分析してもらいます。
最初に **5 ドル（約 750 円）** チャージすれば、 月数十回の分析ができます。

### 手順

1. 🌐 <https://platform.openai.com/> にアクセス
   - **注意**：`chatgpt.com` ではありません！別サイトです
2. 「Sign up」または、ChatGPT を使っているなら同じアカウントで「Log in」
3. 左メニュー「**Settings → Billing**」
4. 「**Add payment method**」でクレカを登録
5. 「**Add to credit balance**」で **5 ドル** をチャージ
   - 「Auto recharge」は **オフ** のままにしておくと安心です
6. 左メニュー「**API keys**」を開く
7. 「**+ Create new secret key**」をクリック
8. 入力画面：
   - **Name**：`ig-reel-research`
   - **Project**：`Default project` のまま
   - **Permissions**：**「All」** を選ぶ
9. 「**Create secret key**」をクリック
10. 表示された `sk-proj-xxxxxxxxxxxxx...` を **必ずコピーしてメモ**

> 🛑 **このキーもこの画面でしか表示されません**。コピー必須。

## Phase 3: Google Cloud の自動操作アカウントを作る（20 分）

ここが一番手間ですが、 **手順通りクリックするだけ** で完了します。

### Google Cloud って何？

Google が提供する **クラウドサービス** です。今回はスプレッドシートに自動で書き込むための「**自動操作アカウント**」を作ります。 **すべて無料の枠で完結します**。

### 3-1. 新しいプロジェクトを作る（3 分）

1. 🌐 <https://console.cloud.google.com/> にアクセス（普段使う Google アカウントでログイン）
2. 画面上部のドロップダウン（プロジェクト選択）をクリック
3. 右上の「**新しいプロジェクト**」をクリック
4. プロジェクト名：`ig-reel-research-自分の名前` のように分かりやすい名前に
5. 「**作成**」をクリック → 数十秒で完成
6. 上部のドロップダウンで作ったプロジェクトを選択

> 📸 [スクショ: `screenshots/3a_gcp_project_create.png`]

### 3-2. 必要なサービスを 2 つ「ON」にする（2 分）

1. 左上のハンバーガー（≡）メニュー → 「**API とサービス → ライブラリ**」
2. 検索ボックスに `Google Sheets API` と入力
3. 出てきた「Google Sheets API」をクリック → 「**有効にする**」
4. もう一度ライブラリに戻り、`Google Drive API` を検索 → 同じく「**有効にする**」

> 📸 [スクショ: `screenshots/3b_gcp_apis_enabled.png`]

### 3-3. 自動操作アカウントを作る（5 分）

1. 左上のハンバーガー（≡）メニュー → 「**IAMと管理 → サービスアカウント**」
2. 上部の「**+ サービスアカウントを作成**」をクリック
3. 名前：`sheets-writer`
4. 「**作成して続行**」
5. ロール選択画面：
   - 「ロールを選択」のドロップダウンを開く
   - 検索欄に `編集者` と入力
   - 「**基本 → 編集者**」を選ぶ
6. 「**続行**」→ 「**完了**」

> 📸 [スクショ: `screenshots/3c_service_account_role.png`]

### 3-4. パスワード（鍵）ファイルをダウンロード（3 分）

1. 作ったサービスアカウント（`sheets-writer@...` という行）をクリック
2. 上部のタブで「**キー**」を選ぶ
3. 「**鍵を追加 → 新しい鍵を作成**」
4. キーのタイプ：**JSON** を選択 → 「**作成**」
5. ファイルが自動ダウンロードされます（`xxx-yyyy.json` みたいな名前）

> 📸 [スクショ: `screenshots/3d_create_key_json.png`]

### 3-5. ダウンロードしたファイルを所定の場所に移動（2 分）

ターミナルに戻って以下を実行（実際のファイル名に置き換えてください）：

```bash
# Downloads から移動（ファイル名はあなたのものに置き換え）
mv ~/Downloads/プロジェクト名-xxxxxxxxxx.json ~/.config/gcloud/sheets-writer.json

# 安全のためのアクセス制限
chmod 600 ~/.config/gcloud/sheets-writer.json
```

> 💡 ファイル名がわからなければ Finder で確認するか、`ls -lt ~/Downloads/*.json` で最新のものを確認できます。

### 3-6. 自動操作アカウントのメールをコピー（1 分）

ダウンロードした設定ファイルの中に書かれているメールアドレス（`sheets-writer@xxx.iam.gserviceaccount.com` のような形）が必要です。

ターミナルでこれを実行：

```bash
python3 -c "import json; print(json.load(open('/Users/$(whoami)/.config/gcloud/sheets-writer.json'))['client_email'])"
```

→ 表示されたメールアドレスを **コピーしてメモ**（次の Phase 4 で使います）

## Phase 4: Google スプレッドシートを準備（5 分）

### 手順

1. 🌐 <https://sheets.google.com/> で「**空白のスプレッドシート**」を新規作成
2. タイトル（左上）を `IG Reel Research` などに変更
3. 右上の **「共有」** ボタンをクリック
4. 入力欄に **Phase 3-6 でメモしたメールアドレス** を貼り付け
5. 権限：**編集者** に設定
6. 「通知を送信」のチェックを **外す**（自動操作アカウントは人ではないので）
7. 「**共有**」または「**送信**」をクリック
8. ブラウザのアドレスバーから **スプレッドシートID** をコピー

   URL の形式はこうなっています：
   ```
   https://docs.google.com/spreadsheets/d/【ここをコピー】/edit
   ```

## Phase 5: 取得した 4 つの情報を Mac に登録（10 分）

これでメモした情報は 4 つあるはずです：

| メモした情報 | 例 |
|---|---|
| Apify トークン | `apify_api_xxxxxxxxxxxxxxx...` |
| OpenAI の鍵 | `sk-proj-xxxxxxxxxxxxxxx...` |
| 自動操作アカウントの設定ファイル | `~/.config/gcloud/sheets-writer.json` |
| スプレッドシート ID | `1AbC2dEfGhI...` |

これらを Mac に登録します。

### 5-1. このプロジェクトを所定の場所にコピー

```bash
# このプロジェクトをダウンロードしたフォルダに移動
# （例：~/Downloads/ig-reel-research）

# Mac の AI ツール用フォルダにコピー
cp -r ig-reel-research ~/.claude/skills/

# 確認
ls ~/.claude/skills/ig-reel-research/
```

### 5-2. スプレッドシート ID を設定

ターミナルで以下を実行（**`1AbC2dEfGhI...` の部分を自分の ID に置き換え**）：

```bash
# 例：あなたのスプシIDが 1AbC2dEfGhI... の場合
sed -i '' "s|YOUR_SPREADSHEET_ID_HERE|1AbC2dEfGhI...|" ~/.claude/skills/ig-reel-research/scripts/gsheets.py

# 設定確認
grep "DEFAULT_SPREADSHEET_ID" ~/.claude/skills/ig-reel-research/scripts/gsheets.py
```

### 5-3. Mac の初期設定ファイルに情報を追記

ターミナルで：

```bash
open -a "TextEdit" ~/.zshrc
```

開いた TextEdit の **末尾** に以下を **そのままコピペ** して、引用符の中身を実際の値に置き換えます：

```bash
# ig-reel-research
export APIFY_TOKEN="ここに Apify トークンを貼り付け"
export OPENAI_API_KEY="ここに OpenAI の鍵を貼り付け"
export GOOGLE_SHEETS_CREDENTIALS="$HOME/.config/gcloud/sheets-writer.json"
```

保存（**⌘ + S**）して TextEdit を閉じる。

> ⚠️ 「リッチテキスト形式」になりそうな場合は **「フォーマット → 標準テキストにする」** を先に選んでから保存してください。

### 5-4. 設定を反映

ターミナルで：

```bash
source ~/.zshrc
echo "APIFY=${#APIFY_TOKEN}文字 / OPENAI=${#OPENAI_API_KEY}文字 / JSON=$GOOGLE_SHEETS_CREDENTIALS"
```

下のような表示になればOK：
```
APIFY=46文字 / OPENAI=164文字 / JSON=/Users/yourname/.config/gcloud/sheets-writer.json
```

3 つすべての値が入っていれば成功です！

## Phase 6: 動作確認（10 分）

ここまで来たら、Claude Code を一度終了して、再起動してください。

その後、Claude Code のチャットでこう打ちます：

```
ダイエットでリサーチして
```

すると AI から条件をヒアリングされるので、 **キーワードだけ** 答えればOK：

```
ダイエット
```

→ あとは 15〜30 分待つだけ。完了すると **「リサーチ完了！○件収集→○件分析。」** と表示されます。

> 📸 [スクショ: `screenshots/6_test_run_complete.png`]

スプレッドシートを開くと、3 つのタブができているはずです：

- 「**使い方ガイド**」 — このツールの仕様の説明
- 「**ウォッチリスト**」 — お気に入りアカウント登録用（最初は空）
- 「**ダイエット**」 — リサーチ結果のまとめ ⭐

ここまで終われば **セットアップ完了** です！
